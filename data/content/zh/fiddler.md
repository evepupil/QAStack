# Fiddler Everywhere

Fiddler Everywhere 是一款跨平台的 Web 调试代理工具，帮助开发者调试和分析任何浏览器、系统或平台的 HTTP/HTTPS 流量。

## 主要特点

- **跨平台** - 支持 macOS、Windows 和 Linux
- **请求检查** - 查看详细的请求/响应数据
- **自动响应** - 自动 Mock API 响应
- **API 构建器** - 构建和测试 API 请求
- **流量对比** - 对比捕获的会话
- **团队协作** - 与团队成员分享会话

## 快速开始

1. 从官网下载 Fiddler Everywhere
2. 启动应用
3. 启用系统代理捕获
4. 开始浏览以捕获流量

创建自动响应规则：

```
匹配: EXACT:https://api.example.com/users
动作: 返回文件 → mock-users.json
```
