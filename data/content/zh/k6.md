# k6

k6 是专为开发者设计的现代化负载测试工具，使用 JavaScript 编写测试脚本，用 Go 引擎运行。

## 主要特点

- **JavaScript 测试** - 使用熟悉的 JavaScript/TypeScript 编写
- **高性能** - 基于 Go 的引擎执行效率高
- **CI/CD 友好** - 易于集成到任何 CI/CD 流水线
- **多协议** - 支持 HTTP、WebSocket、gRPC 等
- **扩展** - 通过 xk6 扩展功能
- **云集成** - 可选的 Grafana Cloud 执行

## 快速开始

安装 k6：

```bash
# macOS
brew install k6

# Windows
choco install k6

# Docker
docker run -i grafana/k6 run - <script.js
```

创建测试脚本 `script.js`：

```javascript
import http from 'k6/http';
import { check, sleep } from 'k6';

export const options = {
  vus: 10,          // 虚拟用户数
  duration: '30s',  // 持续时间
};

export default function () {
  const res = http.get('https://test-api.k6.io');
  check(res, {
    '状态码是 200': (r) => r.status === 200,
    '响应时间 < 500ms': (r) => r.timings.duration < 500,
  });
  sleep(1);
}
```

运行测试：

```bash
k6 run script.js
```

## 为什么选择 k6？

- **开发者友好** - 代码化测试，非 GUI 点击
- **高性能** - Go 语言编写，性能出众
- **现代化** - 专为微服务和云原生应用设计
- **Grafana 集成** - 开箱即用的精美仪表板
