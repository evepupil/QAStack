# MeterSphere

MeterSphere 是一站式开源企业级持续测试平台，涵盖测试跟踪、接口测试、UI 测试和性能测试。

## 主要特点

- **测试管理** - 全面的测试用例管理
- **接口测试** - REST API 测试与自动化
- **性能测试** - 基于 JMeter 的负载测试
- **UI 测试** - 基于 Selenium 的 Web UI 测试
- **CI/CD 集成** - Jenkins、GitLab CI 集成
- **团队协作** - 多租户支持

## 快速开始

使用 Docker 部署：

```bash
docker-compose up -d
```

或使用快速安装脚本：

```bash
curl -sSL https://resource.fit2cloud.com/metersphere/metersphere/releases/latest/download/quick_start.sh | bash
```

访问平台：`http://localhost:8081`

## 为什么选择 MeterSphere？

- **一站式** - 一个平台完成所有测试
- **开源** - 免费使用和定制
- **企业级** - 角色权限控制、审计日志
- **国产** - 优秀的中文文档支持

## 定价

- **社区版** - 免费开源
- **企业版** - 商业支持和额外功能
