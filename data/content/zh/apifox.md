# Apifox

Apifox 是一体化 API 协作平台，集成 API 文档、调试、Mock 和自动化测试功能，相当于 Postman + Swagger + Mock + JMeter。

## 主要特点

- **API 文档** - 美观、交互式的 API 文档
- **API 调试** - 强大的请求调试功能
- **Mock 服务** - 智能 Mock 数据生成
- **自动化测试** - 支持断言的 API 测试自动化
- **团队协作** - 实时协作功能
- **导入导出** - 支持 OpenAPI、Postman 等格式

## 快速开始

1. 从官网下载 Apifox
2. 创建新项目
3. 导入 API 规范或手动创建接口
4. 开始调试和测试

API 请求示例：

```
请求方法: POST
URL: https://api.example.com/users
请求头:
  Content-Type: application/json
请求体:
{
  "name": "张三",
  "email": "zhangsan@example.com"
}
```

## 为什么选择 Apifox？

- **一体化** - 替代 Postman + Swagger + Mock + JMeter

- **中文支持** - 对中文团队友好
