# Charles Proxy

Charles Proxy 是一款强大的 HTTP 代理/抓包/监视工具，可以查看设备与互联网之间的所有 HTTP 和 HTTPS 流量。

## 主要特点

- **SSL 代理** - 以明文查看 SSL 请求和响应
- **弱网模拟** - 模拟慢速网络条件
- **断点** - 拦截和编辑请求/响应
- **请求重写** - 发送前修改请求
- **流量录制** - 记录所有 HTTP 流量用于分析
- **Map Local/Remote** - 将请求重定向到本地文件或其他服务器

## 快速开始

1. 从官网下载安装 Charles
2. 配置设备使用 Charles 作为代理
3. 安装 Charles 根证书用于 HTTPS 抓包
4. 开始录制流量

配置 SSL 代理：

```
代理 → SSL 代理设置 → 添加
主机: *
端口: 443
```
