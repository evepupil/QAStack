# Burp Suite

Burp Suite is the world's most widely used web security testing toolkit, providing everything you need for web application security testing.

## Key Features

- **Proxy**: Intercept and modify HTTP/HTTPS traffic
- **Scanner**: Automated vulnerability scanning
- **Intruder**: Automated customized attacks
- **Repeater**: Manual request manipulation
- **Sequencer**: Token and session analysis
- **Decoder**: Encoding/decoding utilities
- **Comparer**: Visual data comparison

## Quick Start

1. Download Burp Suite from PortSwigger
2. Configure your browser to use Burp as proxy (127.0.0.1:8080)
3. Install Burp's CA certificate for HTTPS interception
4. Start intercepting and analyzing traffic

Intercept a request:

```
Proxy → Intercept → Intercept is on
Browse to target website
Modify request in Burp
Forward to continue
```
