# k6

k6 is a modern load testing tool built for developers, allowing you to write tests in JavaScript and run them with a Go-based engine.

## Key Features

- **JavaScript Tests**: Write tests in familiar JavaScript/TypeScript
- **High Performance**: Go-based engine for efficient execution
- **CI/CD Ready**: Easy integration with any CI/CD pipeline
- **Protocols**: HTTP, WebSocket, gRPC, and more
- **Extensions**: Extend functionality with xk6 extensions
- **Cloud Integration**: Optional cloud execution with Grafana Cloud

## Quick Start

Install k6:

```bash
# macOS
brew install k6

# Windows
choco install k6

# Docker
docker run -i grafana/k6 run - <script.js
```

Create a test script `script.js`:

```javascript
import http from 'k6/http';
import { check, sleep } from 'k6';

export const options = {
  vus: 10,
  duration: '30s',
};

export default function () {
  const res = http.get('https://test-api.k6.io');
  check(res, {
    'status is 200': (r) => r.status === 200,
    'response time < 500ms': (r) => r.timings.duration < 500,
  });
  sleep(1);
}
```

Run the test:

```bash
k6 run script.js
```

## Why Choose k6?

- **Developer Friendly**: Code-based tests, not GUI clicking
- **Fast**: Written in Go for high performance
- **Modern**: Built for microservices and cloud-native apps
- **Grafana Integration**: Beautiful dashboards out of the box
