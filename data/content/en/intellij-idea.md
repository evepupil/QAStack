# IntelliJ IDEA

IntelliJ IDEA is the ultimate IDE for Java developers, offering unmatched intelligent coding assistance and perfect integration with testing frameworks like JUnit, TestNG, and Selenium.

## Key Features

- **Smart Completion**: Context-aware code completion
- **Refactoring**: Safe, automated code refactoring
- **Built-in Tools**: Debugger, profiler, database tools
- **Framework Support**: Spring, Jakarta EE, Micronaut
- **Testing Integration**: JUnit, TestNG, Spock support
- **Version Control**: Git, SVN, Mercurial integration

## Quick Start

Download from jetbrains.com and install.

Run tests directly in IDE:
- Right-click on test class â†’ Run
- Use gutter icons to run individual tests
- View results in Run tool window

Configure test framework:

```xml
<!-- pom.xml -->
<dependencies>
    <dependency>
        <groupId>org.junit.jupiter</groupId>
        <artifactId>junit-jupiter</artifactId>
        <version>5.10.0</version>
        <scope>test</scope>
    </dependency>
</dependencies>
```

Useful shortcuts:
- `Ctrl+Shift+T`: Create/Go to test
- `Ctrl+Shift+F10`: Run current test
- `Ctrl+F5`: Rerun last test
- `Alt+Enter`: Quick fixes
