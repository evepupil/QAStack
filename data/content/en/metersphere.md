# MeterSphere

MeterSphere is an open-source enterprise-class continuous testing platform that covers test tracking, API testing, UI testing, and performance testing.

## Key Features

- **Test Management**: Comprehensive test case management
- **API Testing**: REST API testing with automation
- **Performance Testing**: Load testing based on JMeter
- **UI Testing**: Selenium-based web UI testing
- **CI/CD Integration**: Jenkins, GitLab CI integration
- **Team Collaboration**: Multi-tenant support

## Quick Start

Deploy with Docker:

```bash
docker-compose up -d
```

Or use the quick install script:

```bash
curl -sSL https://resource.fit2cloud.com/metersphere/metersphere/releases/latest/download/quick_start.sh | bash
```

Access the platform at `http://localhost:8081`
