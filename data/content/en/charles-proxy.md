# Charles Proxy

Charles Proxy is a web debugging proxy application that enables developers to view all HTTP and HTTPS traffic between their machine and the Internet.

## Key Features

- **SSL Proxying**: View SSL requests and responses in plain text
- **Bandwidth Throttling**: Simulate slow network conditions
- **Breakpoints**: Intercept and edit requests/responses
- **Request Rewriting**: Modify requests before they're sent
- **Recording**: Record all HTTP traffic for analysis
- **Map Local/Remote**: Redirect requests to local files or other servers

## Quick Start

1. Download and install Charles from the official website
2. Configure your device to use Charles as proxy
3. Install Charles root certificate for SSL proxying
4. Start recording traffic

Configure SSL Proxying:

```
Proxy → SSL Proxying Settings → Add
Host: *
Port: 443
```
