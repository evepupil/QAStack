# Fiddler Everywhere

Fiddler Everywhere is a web debugging proxy tool that helps developers debug and analyze HTTP/HTTPS traffic across any browser, system, or platform.

## Key Features

- **Cross-Platform**: Works on macOS, Windows, and Linux
- **Request Inspection**: View detailed request/response data
- **Auto Responder**: Mock API responses automatically
- **API Composer**: Build and test API requests
- **Traffic Comparison**: Compare captured sessions
- **Team Collaboration**: Share sessions with team members

## Quick Start

1. Download Fiddler Everywhere from the official website
2. Launch the application
3. Enable system proxy capture
4. Start browsing to capture traffic
